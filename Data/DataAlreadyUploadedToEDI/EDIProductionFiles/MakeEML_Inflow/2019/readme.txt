Description of inflow data files in this folder:

inflow_for_EDI_2013_2017 - missing pressure data and flows are left as NAs and not interpolated

inflow_interpolation_2013_2017 - missing pressure data are linearly interpolated and flow is calculated accordingly - FOR LAB USE ONLY!

inflow_working - most up-to-date file if you want late-breaking flows!; as of 24OCT18, includes cleaned and interpolated data from 2013-2017 and not-finalized data from 2018
